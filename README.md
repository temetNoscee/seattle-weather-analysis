# Seattle Weather Analysis

## Overview

This repository contains an exploratory data analysis (EDA) project completed during my university studies. The project analyzes the **Seattle Weather** dataset from Kaggle to explore weather patterns and relationships between precipitation, temperature, wind, and weather conditions.

The objective of this project was to practice data exploration, visualization, and statistical analysis using Python.

> **Note:** This is an academic project that I keep as part of my learning journey. It does not reflect my current professional focus.

---

## Dataset

**Source:** https://www.kaggle.com/datasets/ananthr1/weather-prediction

The dataset contains daily weather observations recorded in Seattle between **2012 and 2015**.

### Features

| Column | Description |
|---------|-------------|
| Date | Observation date |
| Precipitation | Daily precipitation (inches) |
| Temp Max | Maximum daily temperature |
| Temp Min | Minimum daily temperature |
| Wind | Average wind speed |
| Weather | Weather condition (rain, sun, drizzle, snow, fog) |

These visualizations were used to better understand the relationships between weather variables and identify general patterns in the dataset.

---

## Key Findings

- The dataset contains no missing values.
- Temperature variables show a strong positive correlation.
- Most days have little or no precipitation, while heavy rainfall occurs less frequently.
- Weather categories exhibit distinct distributions across temperature and precipitation values.
- Wind speed varies across weather conditions but remains within a relatively limited range for most observations.
